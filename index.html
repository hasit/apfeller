AI-powered shell apps

apfeller

Small shell apps built on apfel. Runs fully local on your Mac, zero API cost, no API keys, no cloud round-trip.

The AI apps keep inputs and outputs small for instant single-turn results inside apfel's fixed 4,096-token combined window. port is the one simple non-AI local utility.

Install

curl -fsSL https://raw.githubusercontent.com/hasit/apfeller/main/install.sh | sh

AI apps

  • cmd — natural language to shell command
  • oneliner — plain English to pipe chain
  • define — tiny dictionary lookup

Local utility

  • port — see what owns a local port
